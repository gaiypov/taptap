# Создание тестовых видео через SQL

Если скрипт `npm run seed:videos` не работает из-за проблем с сетью/DNS, используйте этот SQL скрипт.

## Инструкция:

1. Откройте **Supabase Dashboard**
2. Перейдите в **SQL Editor**
3. Скопируйте содержимое файла `scripts/create-test-videos.sql`
4. Вставьте в SQL Editor
5. Нажмите **Run**

## Что создаст скрипт:

- ✅ Тестовый пользователь (если его нет)
- ✅ 3 автомобиля с видео
- ✅ 3 лошади с видео
- ✅ 3 недвижимости с видео
- ✅ Все детали (car_details, horse_details, real_estate_details)

## Проверка:

После выполнения скрипта вы увидите таблицу с количеством созданных записей по категориям.

Все записи будут со статусом `published` и сразу появятся в фиде приложения.
