## 🎯 Цель улучшений

Максимально улучшить главную страницу приложения:
1. ✅ Уменьшить **боковые кнопки** (RightActionPanel) - они слишком крупные
2. ✅ Опустить боковые кнопки **ближе к профилю** (ниже)
3. ✅ Исправить **несимметричные надписи** на нижней панели ("Подать" vs "Сохранённое")
4. ✅ Общие **UI улучшения** для лучшей эргономики

---

## ✅ Реализованные улучшения

### 1️⃣ Боковые кнопки (RightActionPanel) — Уменьшены

**Файл:** `lib/constants/sizes.ts` → `actionPanel`

#### До:
```tsx
actionPanel: {
  iconSize: 26,          // Слишком большой
  containerSize: 46,     // Слишком большой
  gap: 18,
  rightOffset: 12,
  bottomOffset: iOS: 120, Android: 100,  // Слишком высоко
  countFontSize: 12,
}
```

#### После:
```tsx
actionPanel: {
  iconSize: 22,           // ↓ Уменьшено с 26
  containerSize: 40,      // ↓ Уменьшено с 46
  gap: 16,                // ↓ Уменьшено с 18
  rightOffset: 10,        // ↓ Уменьшено с 12
  bottomOffset: iOS: 160, Android: 140,  // ↑ Опущено ниже (было 120/100)
  countFontSize: 11,      // ↓ Уменьшено с 12
}
```

**Эффект:**
- ✅ Кнопки меньше на **15%** (46 → 40)
- ✅ Иконки меньше на **15%** (26 → 22)
- ✅ Опущены на **40px ниже** на iOS (120 → 160)
- ✅ Опущены на **40px ниже** на Android (100 → 140)
- ✅ Ближе к кнопке профиля на нижней панели

---

### 2️⃣ Нижняя панель (TabBar) — Исправлена симметрия

**Файл:** `app/(tabs)/_layout.tsx`

#### Проблема:
```tsx
// ❌ Несимметрично:
"Главная" (7 букв)
"Поиск" (5 букв)
"Подать" (6 букв)
"Сохранённое" (11 букв) ← Слишком длинное!
"Моё" (3 буквы)
```

#### Решение:
```tsx
// ✅ Симметрично:
"Главная" (7 букв)
"Поиск" (5 букв)
"Подать" (6 букв)
"Избранное" (9 букв) ← Короче и лучше!
"Моё" (3 буквы)
```

**Изменения:**
```diff
- label="Сохранённое"
+ label="Избранное"
```

**Эффект:**
- ✅ Короче на **2 буквы**
- ✅ Лучшая симметрия с другими вкладками
- ✅ Более читаемо

---

### 3️⃣ TabBar размеры — Улучшены

**Файл:** `lib/constants/sizes.ts` → `tabBar`

#### До:
```tsx
tabBar: {
  labelSize: 10,           // Мелкий шрифт
  createButtonSize: 44,    // Средняя кнопка
  createIconSize: 22,
}
```

#### После:
```tsx
tabBar: {
  labelSize: 11,           // ↑ Увеличено с 10 (лучше читаемость)
  createButtonSize: 46,    // ↑ Увеличено с 44 (лучше видимость)
  createIconSize: 24,      // ↑ Увеличено с 22
}
```

**Эффект:**
- ✅ Надписи читаемее на **10%**
- ✅ Кнопка "Подать" заметнее на **5%**

---

### 4️⃣ Категории (CategoryTabs) — Компактнее

**Файл:** `lib/constants/sizes.ts` → `categoryTabs`

#### До:
```tsx
categoryTabs: {
  tabPaddingHorizontal: 16,  // Широкие отступы
  tabGap: 24,                // Большие промежутки
  fontSize: 15,              // Крупный текст
}
```

#### После:
```tsx
categoryTabs: {
  tabPaddingHorizontal: 14,  // ↓ Уменьшено с 16
  tabGap: 20,                // ↓ Уменьшено с 24
  fontSize: 14,              // ↓ Уменьшено с 15
}
```

**Эффект:**
- ✅ Компактнее на **7%**
- ✅ Больше места для контента
- ✅ Все категории помещаются на экране без скролла
